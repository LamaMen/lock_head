# Система автоматической выдачи материалов для рабочих
 Вся система состоит из двух компонентов:  сервера и ардуино.

### Сервер
Сервер отвечает за управление состоянием системы. Он отвечает на запросы от клиентской части, обрабатывает данные датчиков, работает с базой данных.

Сервер написан на языке Python, c использованием фреймворка Flask. В качестве СУБД используется SQLite. Все необходимые для запуска зависимости находятся в файле `requirements.txt`. Так как для работы серверу необходима связь с другими компонентами, в проекте есть файлы, которые используют специфичные для *Raspberry Pi* библиотеки, например *RPI.GPIO* для управления выходами платы и *smbus* для обеспечения *I2C* связи с *Arduino*. **Поэтому во время тестового запуска на компьютере, отличающимся от *Raspberry Pi* могут возникать ошибки.** Для предотвращения данной ситуации рекомендуется комментировать проблемные места, а работу с компонентами тестировать непосредственно на *Raspberry Pi*.

### Arduino 
Arduino отвечает за открытие ящиков. Для этого к ней подключены реле. Когда на ардуино приходит сигнал для открытия ящика, она подаёт напряжение по пин соответствующий необходимому реле, оно замыкается и на замок подаётся напряжение в 12 V. Продолжительность подачи напряжения равна 5 секундам. 
* [Описание работы Arduino Mega](./documentation/arduino.md)
